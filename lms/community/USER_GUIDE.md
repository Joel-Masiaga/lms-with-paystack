# Community Messaging - User Guide

## Overview
The Kuza Ndoto Academy platform now includes integrated community features allowing users to communicate, collaborate, and share updates.

## Features

### 1. Direct Messages
Send private one-on-one messages to other users.

**Access:** Sidebar → Community → Messages

**How to Use:**
- Click "Messages" in the Community menu
- View all active conversations
- Click on a user to open the conversation
- Type your message and click "Send"
- Messages are marked as read when viewed

**Features:**
- Automatic conversation creation between two users
- Read status tracking
- Chronological message ordering
- User profile pictures in conversation list

### 2. Community Groups
Join or create groups to discuss topics and connect with like-minded people.

**Access:** Sidebar → Community → Groups

**How to Use - Joining Groups:**
- Click "Groups" in the Community menu
- Browse "Public Groups" section
- Click "Join Group" to become a member
- Once joined, message appears in "Your Groups"
- Click "View Group" to see messages and members

**How to Use - Creating Groups:**
- Click "Create Group" button in the Groups section
- Fill in group name and description
- Choose "Public" (anyone can find and join) or "Private" (invite-only)
- Click "Create Group"
- You become the group admin automatically
- Other users can find and join public groups

**Features:**
- Public/private group types
- Member list with admin roles
- Group chat messages
- Group member count
- Creator information

### 3. Course Broadcasts
**[Instructors Only]** Send announcements to all participants in your courses.

**Access:** Sidebar → Community → Broadcast

**How to Use:**
- Click "Broadcast" in the Community menu (instructors only)
- Select a course from your courses
- Enter subject line
- Write broadcast message
- Click "Send Broadcast"
- All enrolled students will see it in the course broadcast list

**Features:**
- Broadcast only to your own courses
- Subject line for organization
- Recipients are all course enrollees
- Broadcast history visible to students

## Tips for Best Experience

1. **Make Your Profile Complete** - Add a profile picture so others can recognize you
2. **Be Respectful** - Community is for constructive communication
3. **Use Groups Strategically** - Create groups for specific interests or study topics
4. **Check Messages Regularly** - New messages will appear in the Community menu
5. **Descriptive Group Names** - Help others find groups with clear names

## Troubleshooting

**Q: Can't see the Community menu?**
A: You must be logged in. Go to Home → Login

**Q: Why can't I post in a group?**
A: You need to join the group first. Click the "Join Group" button.

**Q: How do I leave a group?**
A: View the group and click "Leave" button in the header.

**Q: Where are my conversations?**
A: Go to Community → Messages to see all your active conversations.

**Q: Can I delete a message?**
A: Not yet - we're working on message editing/deletion features for future releases.
